## ⚠️ 三件事一定要記得（CLAUDE.md 也有寫）

### 1. 圈速分析必先過濾

```javascript
const clean = laps.filter(l => l.IsAccurate === true);
```

不過濾會混入進站圈、出站圈、安全車圈，整張圖會被尖刺毀掉。

### 2. LapTime 要轉成秒數

```javascript
import { parseTimeToSeconds } from '@/lib/utils';
const sec = parseTimeToSeconds(row.LapTime);
```

原始格式是 timedelta 字串，不能直接做數學運算。

### 3. stints 要 join results 才有車手代碼

```javascript
const r = results.find(r => r.DriverNumber === s.driver_number);
```

stints 表只有車號（如 1、4），沒有 VER / NOR。

> **不用背！** 起手式 Prompt 已經幫你寫好了，AI 會自動套用。

## 🆘 卡住了怎麼辦

### 狀況 1：畫面變空白

打開瀏覽器 DevTools（按 **F12**）→ 看 Console 分頁有沒有紅字。

把紅字 **整段複製** 貼給 Claude Code：

```
我的畫面變空白，瀏覽器 console 顯示：

[貼紅字]

請告訴我哪一行錯、怎麼修。
```

### 狀況 2：AI 改完不是我要的

不要回頭修，直接告訴 AI：

```
你剛才的修改我不喜歡 [具體哪裡]，請改成 [我要的]。其他保留。
```

### 狀況 3：API Key 沒額度

每組有 $15 上限。如果跑很多次後用完，**舉手叫助教換另一把備用 Key**。

### 狀況 4：完全卡住超過 10 分鐘

**舉手叫助教**。助教會 pair coding 10 分鐘陪你過難關。

## ❓ FAQ

**Q：可以裝其他套件嗎？**
A：不行。已預裝的（React、Tailwind、shadcn、Recharts、Papa Parse）就夠你發揮。

**Q：可以改其他檔案嗎？**
A：建議只改 `src/App.jsx`。其他檔案（`main.jsx`、`index.css`、`tailwind.config.js`）不要動。

**Q：AI 可以做的事我會的話，自己寫比較快？**
A：可以！這是 Vibe Coding 的精神 — **AI 是工具不是答案**。但展示時記得說「這段我自己寫的」，會加分。

**Q：可以在發表時 demo 真實 GPS 動畫嗎？**
A：`locations.csv` 已經幫你準備好（48,000 筆 GPS 點）。如果你的主題用得上、且時間夠，這是**亮點加分項**。
